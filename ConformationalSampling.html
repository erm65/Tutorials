Conformational sampling

Instructions

Run the "OpenEye-Applications Ver" command line


Step 3. First minimization with Szybki

The goal of this step is the setup of a correct and minimized structure that will be the starting point of the next steps. It is important that this structure is careful checked from a chemical point of view and that the file is formally correct. Check bond order, charges, names and all it was discussed before.

Minimization of the mol2 file with default parameters, now the dielectric setup is not important yet.

In the "OpenEye-Applications Ver" command line, run Szybki

Szybki -mpi_np 8 -in mol.mol2 -out mol_sz.mol2 -prefix mol_sz

Step 4. Setup of the final structure file

The file prepared in Step 3 is submitted to Omega for the conformational sampling.

The dielectric constant should be set on the basis of the explored environment. Some useful values are the following:

  • solvent (solv, short name that should be used in the next steps): dielectric constants
  • water (wat): 80
  • dmso (dms): 47
  • methanol (met): 32.6
  • acetonitrile (acn): 21.01
  • chloroform (chl): 4.81
  • 1,4-dioxane (dxa): 2.21
  • cyclohexane (hex): 2.02

For linear compounds in a certain solvent (solv), here water:

oeomega classic -mpi_np 8 -searchff mmff94s -dielectric 80 -ewindow 10 -strict -in mol_sz.mol2 -out mol_wat_oe.mol2 -prefix mol_wat_oe

For cyclic compounds

oeomega macrocycle -mpi_np 8 -dielectric_constant 80 -ewindow 20 -in mol_sz.mol2 -out mol_wat_oe.mol2 -prefix mol_wat_oe

Then the conformations are definitively minimized submitting the Omega the output to Szybki. This software also calculates and writes the final energy of the conformers.

szybki -mpi_np 8 -solv_dielectric 80 -ff mmff94s -solventPB -solventCA 0.005 -prefix mol_wat_fin -in mol_wat_oe.mol2 -out mol_wat_fin.mol2 -report

Furthermore, Szybki writes the final energy of the conformations in a report (extension .rpt) that will bu used for the final report.

Step 5. Table preparation

Run Vega

In Vega: File - Open and select filename_mol_wat_final.mol2

The file mol2 is uploaded in Vega and the "Analysis of the trajectory" window is opened

In the "Analysis of the trajectory" window, select a property (Polar surface area, PSA; Gyration radius and Molecular Lipophilicity Potentials to calculate the Virtual Log P) and click ok button.

Vega calculates and plots the property

Click on "Save as csv file" to save the results.

Import the results in Excel and merge with the energy output of Szybki.
