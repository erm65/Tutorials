BRMatlab

Instructions

Launch Matlab

Donwload the file ProjBRMatlab.zip

  • Ask to GE

Open the zip file in a directory of your choice

  • List of file in the directory after unzip
    • BRMatlab_Elab.m
    • BRMatlab_Prepare.m
    • Directory libPLS_1.98

In Matlab open the directory with the scripts using the window Current Folder and then add the folder and the subfolder to Matlab path using the mouse right bottom

Type in the Command Window

BRMatlab_Prepare

Then choose the descriptor matrix and the response file in the GUI

  • In the two files the  data are separated by ";"
  • Other separators were not tested
  • There are no restrictions about the working directory

BRMatlab_Prepare writes two files in the working directory:

  • BRMatlab_DataMatrix.csv
    • It will be read by BRMatlab_Elaborate.m
    • TrainingTest.txt
      • The file contains the name of the object and a numerical flag
      • In the file the flag is equal to 1
      • Change the flag value to 0 for compounds in the test set

When the test set is defined, type in the Command Window

BRMatlab_Elab

The script runs PLS and performs  BR analysis.

The results are written  in a file named BRMatlab_Results that is located in the directory where the scripts are located  (BUG to be fixed).
